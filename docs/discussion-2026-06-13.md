## 四、书签功能（文件记忆）

### 需求明确

- **范围**：仅 EPUB、PDF（电子书类），图片和 Markdown 不参与
- **记忆内容**：
  1. 最近打开的文件列表（持久化）
  2. 阅读进度（PDF：页码 / EPUB：章节+进度百分比）
  3. 最后打开时间
- **存储**：JSON 文件（`path_provider` 获取应用数据目录）
- **交互**：重启后首页自动恢复上次的文件列表

### 与 PRD 的关系

PRD §4.2 书签功能即为此项。本次实现作为书签 MVP。
